### BitNet and Ternary Weights

**BitNet** is a type of neural network model designed for efficient inference using binary or ternary weights. These models reduce memory usage and computation cost by limiting weight representations to fewer bits (e.g., 1-bit or 3-bit). This can be especially beneficial for deployment on hardware with limited resources.

- **Ternary Weights**: Ternary neural networks use three possible values for their weights, typically -1, 0, and +1. This allows for a balance between model size reduction and maintaining performance compared to traditional full-precision models.
